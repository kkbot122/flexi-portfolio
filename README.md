## My Portfolio - Flexi Lab

A single-page static portfolio website.

## Deploying to Vercel

Import this repository into Vercel and use these project settings:

- Framework Preset: `Other`
- Root Directory: repository root (`./`)
- Build Command: leave empty
- Output Directory: leave empty
- Install Command: leave empty

The root route is configured in `vercel.json`, so Vercel serves
`kisna-kanti-artistic-portfolio.html` at `/`.
